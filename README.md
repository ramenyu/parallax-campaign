# Pixel Website Test

A minimal, dark-themed website featuring subtle pixelated robots and machines with gentle animations.

## Features

- **Pixelated Elements**: Small robots, machines, and drones created with pure CSS
- **Subtle Animations**: Elements move only a few pixels for a non-distracting effect
- **Dark Theme**: Modern dark background with neon accent colors
- **Responsive Design**: Works on desktop and mobile devices
- **Lorem Ipsum Content**: Placeholder text ready to be replaced

## How to Use

Simply open `index.html` in your browser. No build process or dependencies required!

```bash
open index.html
```

## Customization

- **Colors**: Edit the color variables in `style.css`
- **Content**: Replace lorem ipsum text in `index.html`
- **Animation Speed**: Adjust animation duration in CSS keyframes
- **Pixel Elements**: Add more robots/machines by copying existing elements

## Structure

```
pixel-website-test/
├── index.html      # Main HTML structure
├── style.css       # Styles and animations
├── script.js       # Minimal JavaScript for interactivity
└── README.md       # This file
```

## Browser Support

Works in all modern browsers. Uses CSS Grid, Flexbox, and CSS animations.
